Доброго времени суток уважаемые пользователи программы, сейчас я вам расскажу как и где можно применить данный плагин, дам инструкции
Данный плагин предназначен для отрисовки схемы расположения датчиков, но также может быть применен и для других целей.
В данный момент существуют четыре схемы, которые видно при запуске плагина. Прямоугольная, прямоугольная со смещением, радиально-лучевая
и радиально-лучевая со смещением.

1. Как пользоваться

При запуске появится четыре кнопки с выбором нужной вам схемы. В дальнейшем можно будет изменить свой выбор.

После нажатия по кнопке, перед вами предстанут текстовые поля и другие кнопки для ввода параметров.В зависимости от выбранной схемы, 
появившиеся кнопки и вводимые данные будут разные.

После ввода всех данных можно нажать кнопку "ОК", появится новый слой в QGIS со всеми точками.

Для сохранения результата в файл, нужно указать путь, куда он должен быть сохранен. После указания пути нажмите кнопку "ОК" - результат экспортируется.

2. Описание кнопок и данных

	2.1 Описание для прямоугольных схем
	Высота - расстояние от нижней полосы точек, до верхней. Включает в себя координаты точки, то есть крайние точки будут распологаться
	по крайнему значению этого параметра.
	Длина - расстояние от левой полосы точек до правой. Включает в себя координаты точки, то есть крайние точки будут распологаться
	по крайнему значению этого параметра.
	Кол-во вертикальных линий - это ни что иное, как число точек по горизонтали, которые образуют вертикальные линии. Может быть только
	целым и вещественным числом.
	Кол-во горизонтальных линий - это ни что иное, как число точек по вертикали, которые образуют горизонтальные линии. Может быть только
	целым и вещественным числом.
	Поворот - указывается угол, на который нужно повернуть массив точек, вводится в градусах. Причем, поворачиваются точки относительно
	левого нижнего угла, т.е. какой бы угол не был задан, точка в левом нижнем углу всегда будет в одном месте.
	Координаты левого угла 
	х - координата левой нижней точки, от которой будет строиться схема. Задается в соответствии с системой координат, которая выбирается дальше. 
	У - координата левой нижней точки, от которой будет строиться схема. Задается в соответствии с системой координат, которая выбирается дальше. 
	Система координат - выпадающий список с существующими системами координат. Для корректной работы нужно выбрать такую же систему координат
	как в проекте.
	
	2.2 Описание для радиально-лучевых схем
	Радиус - расстояние от центральной точки, до крайней. Включает в себя координаты точки, то есть крайние точки будут распологаться 
	по крайнему значению этого параметра.
	Кол-во линий - количество лучей, исходящих от центра. Лучи находятся друг от друга на равном удалении, то есть между лучами угол одинаков.
	Кол-во точек на линии - количество точек, которые будут распологаться на луче. Самая дальняя точка находится на расстоянии r(радиус)
	между точками дистанция одинаковая. В радиально-лучевой со смещением схеме, точки на нечетных линиях будут смещены. Смещение равно половине дистанции между точками.
	Поворот - указывается угол, на который нужно повернуть массив точек, вводится в градусах. Причем, поворачиваются точки относительно
	центра, т.е. какой бы угол не был задан, точка в центре всегда будет в одном месте.
	Координаты центра 
	х - координата центральной точки, от которой будет строиться схема. Задается в соответствии с системой координат, которая выбирается дальше. 
	У - координата центральной точки, от которой будет строиться схема. Задается в соответствии с системой координат, которая выбирается дальше. 
	Система координат - выпадающий список с существующими системами координат. Для корректной работы нужно выбрать такую же систему координат
	как в проекте.
