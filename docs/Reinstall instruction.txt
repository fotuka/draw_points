Сначала нужно удалить старый плагин, для этого в QGIS:
Модули - Управление модулями - Установленные - "plugin-main" (может быть другое название) - Удалить модуль

Далее скачать новую версию
Перейти по ссылке https://github.com/fotuka/draw_points, нажать на зеленую кнопку "Code", в выпадающем списке выбрать "Download ZIP"

Далее установить новую версию в QGIS:
Модули - Управление модулями - Установить из ZIP файла - выбрать путь к архиву, который скачали ранее - Установить модуль
